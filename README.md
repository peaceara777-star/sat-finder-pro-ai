## 📖 نبذة عن المشروع

<div align="center">

**Sat Finder Pro AI** هو التطبيق العربي الأول والمتكامل لخبراء تركيب وصيانة الأقمار الصناعية.

يجمع بين **الذكاء الاصطناعي** من Google Gemini 1.5 Pro، و **قاعدة بيانات شاملة**، و **أدوات توجيه احترافية** في واجهة عربية أنيقة وسهلة الاستخدام.

</div>
