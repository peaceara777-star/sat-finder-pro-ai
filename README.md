## 📖 نظرة عامة

**Sat Finder Pro AI** هو تطبيق ويب احترافي ومتكامل باللغة العربية، مصمم خصيصاً لخبراء وفنيي الأقمار الصناعية. يجمع التطبيق بين قوة الذكاء الاصطناعي من Google Gemini ودقة مستشعرات الجهاز لتقديم تجربة فريدة في توجيه الأطباق واستكشاف الأخطاء وإصلاحها.

يتميز التطبيق بواجهة زجاجية عصرية (Glassmorphism) مع دعم كامل للغة العربية (RTL)، ويمكن تثبيته كتطبيق مستقل (PWA) على جميع الأجهزة.

### 🌟 لماذا Sat Finder Pro AI؟

- **🧭 بوصلة ذكية حقيقية**: استخدم هاتفك لتوجيه الطبق بدقة متناهية نحو أي قمر صناعي
- **🤖 مساعد ذكي متطور**: احصل على إجابات فورية من خبير الأقمار "سامي" المدعوم بـ Gemini 1.5 Pro
- **🗄️ قاعدة بيانات شاملة**: 500+ تردد، 200+ إصلاح، 50+ ماركة رسيفر، 30+ شريحة معالج
- **📱 تجربة متكاملة**: يعمل على جميع الأجهزة ويمكن تثبيته كتطبيق مستقل

## ✨ المميزات الرئيسية

### 🧭 البوصلة الذكية
- استخدام مستشعرات الجهاز الحقيقية (DeviceOrientation API)
- عرض زوايا السمت (Azimuth) والارتفاع (Elevation) ولف الـ LNB (Skew)
- توجيه بصري وصوتي نحو القمر المستهدف
- محاكاة قوة الإشارة في الوقت الحقيقي
- دعم جميع الأقمار العربية والأوروبية

### 🤖 المساعد الذكي "سامي"
- مدعوم بـ **Google Gemini 1.5 Pro API**
- إجابات احترافية باللغة العربية عن:
  - ترددات القنوات وكيفية استقبالها
  - طرق توجيه الأطباق بدقة
  - حلول المشاكل التقنية والأعطال
  - برمجة الرسيفرات وإعدادات الشيرنج
- وضع البحث المحلي الاحتياطي عند انقطاع الإنترنت
- اقتراحات سريعة للأسئلة الشائعة

### 🗄️ قاعدة بيانات علائقية متكاملة
- **25+** قمر صناعي عربي وأوروبي وآسيوي
- **500+** تردد محدث للقنوات
- **200+** إصلاح موثق للأعطال
- **50+** ماركة رسيفر مع الموديلات الشائعة
- **30+** شريحة معالج مع المواصفات الكاملة
- **25+** أداة برمجة وفلاش وتحرير

### 🔧 دليل الأعطال والصيانة
- تصنيف المشاكل حسب الخطورة (حرج/متوسط/بسيط)
- أسباب المشكلة وحلولها التفصيلية
- الأدوات المطلوبة للإصلاح
- حلول برمجية وهاردوير لكل مشكلة
- نسبة نجاح مقدرة لكل إصلاح

### 📡 مكتبة الترددات
- بحث سريع في الترددات بالاسم أو القيمة
- عرض تفصيلي: التردد، الاستقطاب، معدل الترميز، FEC
- تحديد القنوات المشفرة والمفتوحة
- تصنيف القنوات حسب الفئة (أخبار، رياضة، ترفيه، إلخ)
- نسخ التردد بنقرة واحدة

### 🎨 واجهة مستخدم احترافية
- تصميم زجاجي عصري (Glassmorphism) مع تأثيرات بصرية مذهلة
- دعم كامل للغة العربية (RTL) مع خط Cairo
- تجاوب كامل مع جميع أحجام الشاشات (Mobile First)
- وضع ليلي مدمج
- تأثيرات حركية سلسة (Animations)
- إشعارات Toast جذابة

### ⚙️ إعدادات متقدمة
- إدارة مفتاح Gemini API
- تفعيل/تعطيل المساعد الذكي
- التحكم في الإشعارات
- وضع ليلي/نهاري
- حفظ جميع الإعدادات محلياً

### 📱 تجربة PWA متكاملة
- تثبيت التطبيق على الشاشة الرئيسية
- عمل بدون اتصال بالإنترنت (Offline Mode)
- Service Worker للتخزين المؤقت
- Manifest.json للإعدادات

### 🐳 دعم Docker كامل
- Dockerfile محسن للإنتاج
- docker-compose.yml مع Nginx و Watchtower
- دعم بيئات متعددة (development/production/testing)
- تحديث تلقائي للحاويات

### ⚡ CI/CD جاهز
- GitHub Actions للاختبار والبناء والنشر
- CodeQL للفحص الأمني
- Dependency Review لمراجعة التبعيات
- اختبارات تلقائية على Python 3.9/3.10/3.11 و Node.js 18/20
