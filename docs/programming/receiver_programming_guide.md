## 1. أنواع الشرائح وطرق برمجتها

### 1.1 شرائح Ali (M3511 - M3526)

| الموديل | المعالج | الدقة | سنة الإصدار |
|---------|---------|------|------------|
| M3511 | 800MHz | 1080p | 2015 |
| M3516 | 1.2GHz | 1080p | 2017 |
| M3521 | 1.5GHz | 4K | 2020 |
| M3526 | 2.0GHz | 4K | 2023 |

**طريقة البرمجة عن طريق USB:**
- اسم الملف: update.bin أو factory.bin
- تهيئة الفلاشة: FAT32

**طريقة البرمجة عن طريق RS232:**
- الاداة: AliFlash Tool
- السرعة: 115200 baud

### 1.2 شرائح GX (GX6605 - GX6628)

| الموديل | المعالج | الدقة | سنة الإصدار |
|---------|---------|------|------------|
| GX6605 | 550MHz | 1080p | 2014 |
| GX6621 | 800MHz | 1080p | 2016 |
| GX6628 | 1.2GHz | 1080p | 2018 |

**الاداة:** GXLoader
**ملفات البرمجة:**
- bootloader.bin
- maincode.bin
- logo.bin
- defaultdb.bin

### 1.3 شرائح Amlogic (S905 - S928)

| الموديل | المعالج | الدقة |
|---------|---------|------|
| S905X3 | 1.9GHz | 4K |
| S905X4 | 2.0GHz | 4K |
| S928X | 2.0GHz | 8K |

**الاداة:** USB Burning Tool
**ملف البرمجة:** *.img

## 2. أدوات البرمجة والفلاش

### 2.1 AliFlash Tool

**طريقة الاستخدام:**
1. اختيار المنفذ COM
2. Baud Rate: 115200
3. اختيار bootloader.bin
4. اختيار update.bin
5. Start ثم تشغيل الرسيفر

### 2.2 GXLoader

**طريقة الاستخدام:**
1. اختيار Chip: GX6605/GX6621/GX6628
2. اختيار COM Port
3. تحميل الملفات
4. Download

### 2.3 USB Burning Tool

**طريقة الاستخدام:**
1. File > Import Image > اختيار img
2. Start
3. توصيل USB مع الضغط على Reset

## 3. برمجة البوت لودر

### 3.1 استخراج البوت لودر

**باستخدام AliFlash Tool:**
- Read > Bootloader
- اختر مكان الحفظ

### 3.2 أوامر البوت لودر

**Ali:**
- help - عرض الاوامر
- update - بدء التحديث
- bootm 0x82000000 - إقلاع

**GX:**
- fl info - معلومات الفلاشة
- fl erase - مسح الفلاشة
- go 0x80000000 - تنفيذ الكود
