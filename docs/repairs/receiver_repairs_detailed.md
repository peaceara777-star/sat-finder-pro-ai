## 1. أعطال الباور سبلاي

### 1.1 الجهاز لا يعمل نهائيا

**الاعراض:**
- لا توجد أي استجابة عند الضغط على زر الباور
- لمبة الباور الامامية مطفية تماما
- لا يوجد أي صوت أو صورة

**الاسباب المحتملة:**
1. فيوز الباور محروق
2. تلف في آيسي الباور (Power IC)
3. مكثفات الباور منتفخة أو تالفة
4. قطع في مسار الكهرباء

**خطوات الفحص والاصلاح:**

| الخطوة | الاجراء | الاداة المطلوبة |
|--------|---------|-----------------|
| 1 | فحص كابل الباور واستبداله | كابل باور بديل |
| 2 | قياس الفولت على مخارج الباور سبلاي | ملتيميتر |
| 3 | فحص الفيوز واستبداله | فيوز بديل |
| 4 | فحص المكثفات بصريا | عدسة مكبرة |

**الفولتات الطبيعية:**
- 3.3V للبروسيسور والذاكرة
- 5V للدوائر الرقمية
- 12V للتيونر و LNB
- 24V-30V لـ LNB في بعض الاجهزة

### 1.2 الجهاز يفصل ويعمل بشكل متقطع

**الاعراض:**
- الجهاز يعمل لفترة ثم ينطفئ فجأة
- إعادة تشغيل تلقائية متكررة

**الاسباب:**
1. ارتفاع حرارة المعالج
2. مكثفات باور بدأت في التلف
3. ضعف في آيسي الباور
4. مشكلة في التهوية

**خطوات الاصلاح:**
1. تنظيف فتحات التهوية
2. فحص مروحة التبريد
3. تغيير المعجون الحراري للمعالج
4. استبدال مكثفات الباور

## 2. أعطال البوت والإقلاع

### 2.1 الجهاز معلق على شاشة البداية (Boot Loop)

**الاعراض:**
- الجهاز يعلق على شعار الشركة
- يعيد التشغيل بشكل متكرر
- يظهر ON أو BOOT فقط

**الاسباب:**
1. تلف في ملفات النظام
2. بوت لودر تالف
3. مشكلة في الذاكرة الداخلية
4. RAM تالف

**حلول برمجية:**

| الطريقة | الخطوات | نسبة النجاح |
|---------|---------|------------|
| Recovery Mode | اضغط Menu مع التشغيل | 70% |
| USB Update | حمل سوفتوير على فلاشة | 85% |
| RS232 Flash | استخدم AliFlash Tool | 95% |

### 2.2 رموز الخطأ الشائعة

| الرمز | المعنى | الحل |
|-------|--------|------|
| E001 | خطأ في البوت لودر | إعادة تحميل البوت لودر |
| E002 | خطأ في الذاكرة | فحص RAM |
| E003 | خطأ في الفلاشة | فورمات الفلاشة |
| ERR01 | فشل في التيونر | فحص التيونر |
| ERR02 | فشل في الواي فاي | فحص شريحة الواي فاي |

## 3. أعطال التيونر والإشارة

### 3.1 لا توجد إشارة نهائيا

**الاعراض:**
- قوة الإشارة 0%
- جودة الإشارة 0%

**الاسباب:**
1. تلف في IC التيونر
2. عدم وجود فولت LNB (13V/18V)
3. تلف في مكثفات التيونر

**خطوات الفحص:**

| الخطوة | القياس المتوقع |
|--------|---------------|
| 1 | فحص فولت LNB: 13V عمودي / 18V أفقي |
| 2 | فحص مكثفات التيونر |
| 3 | فحص IC التيونر |

### 3.2 الإشارة ضعيفة أو متقطعة

**الحلول:**
- استبدال مكثفات التيونر: تحسن 20-30%
- LNB عالي الحساسية: تحسن 15-25%
- كابل RG6 عالي الجودة: تحسن 10-15%
- فلتر 5G: تحسن 15-20%

## 4. أعطال الصورة والصوت

### 4.1 لا توجد صورة مع وجود صوت

**الاسباب:**
1. كابل HDMI تالف
2. منفذ HDMI تالف
3. تلف في IC الفيديو

**خطوات الاصلاح:**
1. تجربة كابل HDMI آخر
2. تجربة مخرج AV
3. تغيير دقة العرض (V.Format)
4. فحص IC الفيديو

### 4.2 مشاكل الصوت

| المشكلة | الحل |
|---------|------|
| لا يوجد صوت | فحص التوصيلات |
| صوت مشوش | تغيير الكابل |
| تأخر الصوت | ضبط Audio Delay |
| صوت فقط بدون صورة | استخدام AV |

## 5. أعطال المنافذ

### 5.1 منفذ USB لا يعمل

**خطوات الفحص:**
1. تجربة فلاشة أخرى
2. تهيئة الفلاشة FAT32
3. قياس فولت USB (5V)
4. فحص فيوز USB

### 5.2 منفذ HDMI لا يعمل

**الحلول:**
- استبدال الكابل
- تنظيف المنفذ بالكحول
- إعادة لحام الارجل
- استبدال IC HDMI

## 6. أعطال البرمجة والسوفتوير

### 6.1 مشاكل تحديث السوفتوير

| المشكلة | الحل |
|---------|------|
| فشل التحديث | التأكد من الموديل الصحيح |
| توقف في المنتصف | استخدام فلاشة جديدة |
| Invalid File | rename to update.bin |
