# 📖 دليل المستخدم - Sat Finder Pro AI

## 🚀 البدء السريع

### التثبيت على الهاتف (PWA)
1. افتح التطبيق في متصفح Chrome/Safari
2. اضغط على "تثبيت التطبيق" أو "إضافة إلى الشاشة الرئيسية"
3. سيظهر التطبيق كتطبيق مستقل

### استخدام البوصلة
1. اذهب إلى صفحة "توجيه"
2. اختر القمر المستهدف
3. وجه هاتفك نحو السماء
4. اتبع التعليمات الظاهرة على الشاشة

### البحث عن الترددات
1. اذهب إلى صفحة "ترددات"
2. استخدم شريط البحث
3. اضغط على التردد لنسخه

### استخدام المساعد الذكي
1. اذهب إلى صفحة "المساعد"
2. اكتب سؤالك
3. اضغط إرسال

## ❓ الأسئلة الشائعة

### كيف أحصل على مفتاح Gemini API؟
1. اذهب إلى https://aistudio.google.com/app/apikey
2. سجل الدخول بحساب Google
3. اضغط "Create API Key"
4. انسخ المفتاح وأدخله في الإعدادات

### البوصلة لا تعمل؟
- تأكد من منح إذن المستشعرات
- افتح التطبيق عبر HTTPS (أو localhost)
- بعض الأجهزة القديمة لا تدعم البوصلة

### كيف أحدث الترددات؟
- الترددات تتحدث تلقائياً عند الاتصال بالإنترنت
- يمكنك تحديثها يدوياً من الإعدادات
